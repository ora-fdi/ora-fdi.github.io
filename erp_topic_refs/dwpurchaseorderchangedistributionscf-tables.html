DW_PURCHASE_ORDER_CHANGE_DISTRIBUTIONS_CF

Contains details for Purchase Order Distributions for Accepted Purchase Order Versions. Data for Cancelled Purchase Orders and Cancelled Change Order Versions will not be available. Also, data will appear only when some changes happen for the Purchase Order distributions as part of the Purchase Order Version (Changes may happen for details which are not available on the Fact)
